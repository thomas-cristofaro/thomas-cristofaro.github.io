Robot Rover

Robot creato per l'esame del 5° anno di Tecnico Industriale. E' formato da 14 servomotori, 3 arduini nano, un controller per i motori, una fotocamera, dei sensori vari e un modulo bluetooth. Il software di controllo è scritto in C per arduino.

App Rover

Applicazione Android per comunicare con il Rover via bluetooth. E' scritta in nativo Java per android 2.3. Consente di comandare i motori per il movimento del rover e di ricevere le informazioni dai sensori e dalla fotocamera.
